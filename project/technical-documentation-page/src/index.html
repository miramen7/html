Introduction

Illustrator is an app to edit a picture that can be utilized for general or marketing purpose.

The example of Illustrator use are:

  • Marketing campaign
  • Educational poster
  • Magazine
  • Book
  • Map

With the increasing demand of editing photo in smartphone, Illustrator also creates a mobile app that is available in Google Play and App Store.

Homepage

At homepage, you can take a look and pick a sample that will be the muse for your editing.

You can also upload your editing to become a muse for other users. It also has like feature so you are able to rate the sample that you like the most.

Edit a Picture

To edit a picture, you can sign in and click a camera button on the top right.

Once you are already on editing mode, you can choose the template or go with your own style.

Settings

If you want to change the edit mode layout, you can click settings and choose the layout that fit you the most

You can also click the night mode toggle to activate night mode.

Command

User is also able to perform command from CLI available in the page for easy tweaking.

group command is used to group several pictures project and apply the same editing style to them

exclude command is used if user wants to apply different style to a picture from the pictures grouped

add command is used if user wants to apply the group style to a picture

post command is used if user wants to upload their picture to the sample page

exclude_from_sample command is used if user wants to remove their picture from the sample page

API
